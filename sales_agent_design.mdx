# CRISPR-Tools Sales Agent Design

> **Overview:** A comprehensive sales agent designed to discover, validate, and engage CRISPR researchers through personalized outreach.

## High-Level Architecture

Single Sales Agent implementing two subroutines:

- **🔍 Lead Finding Agent** – discovers and validates new leads
- **📧 Lead Emailing Agent** – drafts, verifies, and sends cold emails

### 🎯 Weekly Targets
- ~50 new target customers emailed per week
- ~50–100 model prompts per week (batched for efficiency)

## 🛠️ Core Components

| Component | Choice for Free/Effective Use |
|-----------|------------------------------|
| **🤖 Model** | **GPT‑4o-mini** (OpenAI free-tier via ChatGPT or free API credits) for lead evaluation, relevance scoring, and email drafting.<br/>*Optionally:* Claude Instant Free or Gemini/Bard free tier for web search and summarization. |
| **🔧 Tools** | • **Google Sheets** – memory / CRM, free<br/>• **Gmail** (separate contact@crispr-tools.com) – email sending<br/>• **Hunter.io free tier** – email discovery<br/>• **MailTester** / free verification tool – validate emails<br/>• **Requests + BeautifulSoup** – scrape About/Contact pages |
| **📝 Instructions** | Detailed prompt instructions for:<br/>• Finding relevant researchers<br/>• Filtering by target fit<br/>• Drafting personalized cold emails<br/>• Formatting output as JSON for Sheets |
| **🛡️ Guardrails** | • Ensure email is professional, polite, safe, and spam-free<br/>• Max 120–150 words per email<br/>• Use personalization (name, organization, role)<br/>• Avoid sensitive or inappropriate content |
## 🔍 Lead Finding Agent Routine

### 1. 🌐 Web/Research Prompting

**GPT‑4o-mini + Gemini/Bard or Claude Instant:**

```prompt
"Find 50 gene editing researchers or professionals in academia, biotech, and startups relevant to CRISPR."
```

**Output structured as JSON:** 
```json
{
  "Name": "string",
  "Org": "string", 
  "Role": "string",
  "Status": "ready"
}
```

### 2. ✅ Lead Verification

Prompt GPT-4o-mini to score leads on relevance:
```prompt
"Would this person find https://www.crispr-tools.com useful?"
```

### 3. 📧 Email Discovery

- Use **Hunter.io free tier** for public email search
- **Fallback:** scrape organization websites (About/Contact pages) using requests + BeautifulSoup
- **Generate naive guesses:** `first.last@org.com` if necessary

### 4. ✔️ Email Validation

Verify using **MailTester** or similar free tools

### 5. 💾 CRM / Memory

Add leads to **Google Sheets** with columns:
```
Name | Org | Role | Email | Status (Status=ready)
```

## 📧 Lead Emailing Agent Routine

### 1. 🎯 Identify Leads Not Emailed

Filter **Google Sheet** where `Status=ready`

### 2. ✍️ Draft Personalized Emails

Prompt **GPT‑4o-mini** to generate a 120-word email per lead:

- Include **Name, Org, Role**
- Introduce **CRISPR-Tools** as free directory
- Output as plain text

### 3. 🛡️ Guardrails

Verify emails are professional, polite, safe.

**Optionally prompt GPT-4o-mini with:**
```prompt
"Check this email and return OK if safe and professional, else rewrite."
```

### 4. 🚀 Send Emails

- Use **Gmail API / Google Apps Script** to send in small batches from `contact@crispr-tools.com`
- Respect Gmail limits (100/day free, 500/day Workspace)

### 5. 📊 Update Sheet

Mark `Status=sent`

## 💡 Implementation Notes / Free Optimization

- **🔄 Batching:** Feed 5–10 leads per prompt to GPT to save tokens
- **🔍 Hybrid Search:** Use Gemini/Bard free-tier for live web research; feed results to GPT‑4o-mini for extraction
- **🛡️ Email Safety:** Always run guardrail prompt before sending
- **📧 Separate email account:** Keep your personal Gmail safe, use `contact@crispr-tools.com`
- **⚠️ Error handling:** Skip leads with invalid emails or duplicates in Sheets
- **🛠️ Manual fallback:** Small-scale scraping + model filtering allows you to run everything without paying for APIs, staying within free-tier limits

> **💡 Key Insight:** This design maximizes free usage while being effective, safe, and automatable.
